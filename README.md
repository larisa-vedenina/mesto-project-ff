# Mesto - Интерактивная галерея фотографий

Интерактивный сервис для публикации фотографий с возможностью редактирования профиля, добавления и удаления карточек.

## Демо

[Открыть проект](https://larisa-vedenina.github.io/mesto-project-ff/)

## Возможности

- Просмотр фотографий в полноэкранном режиме
- Редактирование профиля пользователя
- Добавление новых карточек с фотографиями
- Удаление собственных карточек
- Лайки фотографий
- Валидация форм
- Адаптивный дизайн

## Технологии

- **HTML5** - семантическая верстка
- **CSS3** - Flexbox, Grid, адаптивный дизайн
- **JavaScript (ES6+)** - интерактивность, ООП
- **Webpack** - сборка проекта
- **Babel** - транспиляция JavaScript
- **API** - взаимодействие с сервером

## Функциональность

### Модальные окна
- Редактирование профиля
- Добавление новой карточки
- Обновление аватара
- Подтверждение удаления
- Просмотр изображения

### Валидация форм
- Проверка в реальном времени
- Кастомные сообщения об ошибках
- Блокировка кнопки отправки при ошибках

### Взаимодействие с API
- Загрузка карточек с сервера
- Обновление данных профиля
- Добавление/удаление лайков
- Удаление карточек

## Установка и запуск

### Установка зависимостей
npm install

### Запуск dev-сервера
npm run dev

### Сборка production-версии
npm run build

### Деплой на GitHub Pages
npm run deploy
